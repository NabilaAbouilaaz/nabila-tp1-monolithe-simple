# TP1 - Application Monolithique avec Spring Boot

## Objectif
Réaliser une application monolithique simple avec Spring Boot pour gérer des produits via une API REST CRUD.

## Technologies utilisées
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL / MySQL
- Lombok
- Postman

## Fonctionnalités
- Ajouter un produit
- Afficher tous les produits
- Afficher un produit par ID
- Modifier un produit
- Supprimer un produit

## Lancement
1. Configurer la base de données
2. Modifier application.properties
3. Lancer le projet avec IntelliJ ou Maven
4. Tester avec Postman
